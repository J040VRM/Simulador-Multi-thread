<p align="center">
  <i>🚀 Virtual Memory Paging Simulator implementing page replacement algorithms such as FIFO, LRU, and RANDOM, featuring address translation, process management, and performance statistics.</i>
</p>

---

### 🌟 Features | Funcionalidades

<div align="center">

| Feature | Description                                 | Descrição                                              |
| :-----: | :------------------------------------------ | :----------------------------------------------------- |
|    🧠   | Address translation with paging             | Tradução de endereços com paginação                    |
|    🔄   | FIFO, RANDOM and LRU page replacement       | Substituição de páginas por FIFO, RANDOM e LRU         |
|    👤   | Process creation with page table allocation | Criação de processos com alocação de tabela de páginas |
|    ⚠️   | Page fault detection and handling           | Detecção e tratamento de page faults                   |
|    📊   | Real-time performance statistics            | Estatísticas de desempenho em tempo real               |
|   🖥️   | Interactive terminal menu                   | Menu interativo via terminal                           |
|   👁️   | Physical memory frame visualization         | Visualização dos frames da memória física              |


</div>

### 📦 Dependencies | Dependências


| Tipo      | Dependência | Descrição                                                               |
| --------- | ----------- | ----------------------------------------------------------------------- |
| 🧱 Padrão | `stdio.h`   | Entrada e saída padrão (printf, scanf, etc.)                            |
| 🧱 Padrão | `stdlib.h`  | Alocação dinâmica (`malloc`, `free`, `exit`) e funções utilitárias      |
| 🧱 Padrão | `time.h`    | Geração de números aleatórios (`srand`, `time`) para o algoritmo RANDOM |

### 🧭 Menu Guide | Guia do Menu

| Option | Description                                                 | Descrição                                         |
| ------ | ----------------------------------------------------------------- | ------------------------------------------------------ |
| `1`    | Select the page replacement algorithm: `FIFO`, `RANDOM`, or `LRU` | Seleciona o algoritmo de substituição de páginas       |
| `2`    | Add a new process (PID and size in bytes)                         | Adiciona um novo processo (PID e tamanho em bytes)     |
| `3`    | Simulate memory accesses (PID and virtual address)                | Simula acessos à memória (PID e endereço virtual)      |
| `4`    | Show current state of physical memory frames                      | Mostra o estado atual dos frames da memória física     |
| `5`    | Display simulation statistics                                     | Exibe estatísticas da simulação (acessos, page faults) |
| `0`    | Exit the simulator                                                | Encerra o simulador                                    |
